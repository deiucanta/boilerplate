## Laravel Boilerplate

A starting point for every Laravel project. It will include:

- Assets
    - Bower (done)
    - Grunt (done)
- Deploy
    - Capistrano
- Authentication
    - Login, Register
    - Password Reset
- Admin Panel
    - User Management
- Architecture
    - Folder restructuring (done)
    - PSR-0 Domain Folder (done)
- Base Classes
	- Validator
	- Mailer
	- Composer
	- Repository
	- Observer
- Packages
    - Laravel Generators (done)
